How to get a better location

You MUST have a working network connection.

You MUST have active locations.
Settings > Location : on

You SHOULD use Wifi for getting a location.
Settings > Location > Mode : a setting with Wifi

You SHOULD let the phone use Wifi to scan for location all the time.
Settings > Wifi > Advanced > Scanning always available : on

You SHOULD NOT queue background data (if available).
Settings > Power management > Queue background data : off (option not available on all mobiles)


You COULD TRY turn off the Wifi optimization.
Settings > Wifi > Advanced > Wi-Fi optimization : off

You COULD TRY to turn on Wifi briefly, especially if you don't use Wifi otherwise.
(We've noticed that in some cases this helps to get the location service running.)
